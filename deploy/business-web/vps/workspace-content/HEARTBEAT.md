# Heartbeat Checklist

Run quietly and only surface material changes:

1. Review `content/idea-backlog.md` and `content/publishing-queue.md`.
2. If no solid long-form draft is in progress, create one in `content/articles/`.
3. Update `content/shorts.md` from the strongest current draft.
4. Tighten `business/monetization.md` when you see a practical product angle.
5. Update `business/scoreboard.md` when asset or subscriber counts change.
6. Move content to `status: published` only after explicit human approval.

If there is nothing meaningful to update, reply `HEARTBEAT_OK`.
