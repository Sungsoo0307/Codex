# Heartbeat Checklist

Run this checklist quietly and only surface material changes:

1. Review `content/idea-backlog.md` and `content/publishing-queue.md`.
2. If there is no fresh long-form draft in progress, create one in `content/articles/`.
3. Update `content/shorts.md` with derivative posts from the strongest current draft.
4. Tighten `business/monetization.md` when you see a real product or sponsorship angle.
5. Update `business/scoreboard.md` when the asset count changed.
6. Move only human-approved pieces to `status: published`.

If there is nothing meaningful to update, reply `HEARTBEAT_OK`.
